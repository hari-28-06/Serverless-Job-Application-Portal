# Serverless Job Application Portal

## AWS Services Used

- AWS Lambda
- API Gateway
- DynamoDB
- CloudWatch
- Lightsail VPS
- Nginx

## Features

- View Job Details
- Submit Job Applications
- Store Applications in DynamoDB
- Serverless Backend using Lambda
- API Gateway Integration
- Hosted using Nginx on Lightsail

## Architecture

User
→ Nginx
→ API Gateway
→ Lambda
→ DynamoDB
